Swagger2Apis 快速接入助手

本工具可以帮助您🚀 快速接入 swagger2apis

只需三步即可完成接入:

  1. 选择项目根目录
  2. 填写配置信息
  3. 点击生成按钮

⚠️ 使用前请注意：

  1. 请确保 Git 工作区处于干净状态，以便在需要时可以轻松回退更改。
  2. 生成的配置文件可能需要根据您的项目需求进行调整，请检查并删除不需要的内容。
  3. 请确保项目已安装以下依赖：
    • swagger2apis
    • axios
    • prettier
    您可以选择将它们安装为 dependencies 或 devDependencies。
  4. 注意，这里并不能帮你完成所有的一切，当这里一切完毕后，你还是需要最后一步:书写自己的适配器，关于适配器：查看文档
